# Инструкция по работе с Яндекс картой

## Где находится код карты

Код инициализации карты находится в файле `js/main.js` в функции `initYandexMap()` (примерно строки 550-617).

## Как изменить адрес на карте

### Способ 1: Изменить адрес в коде (рекомендуется)

1. Откройте файл `js/main.js`
2. Найдите функцию `createMap()` внутри `initYandexMap()`
3. Найдите строку с адресом:
   ```javascript
   const address = 'Республика Крым, Симферопольский район, с. Мирное, ул. Центральная, 48';
   ```
4. Измените адрес на нужный вам
5. Сохраните файл

### Способ 2: Использовать координаты напрямую

Если геокодер не находит адрес, можно использовать координаты напрямую:

1. Найдите в коде строку с `ymaps.geocode(address, ...)`
2. Замените весь блок геокодирования на:
   ```javascript
   const coords = [44.9511, 34.1024]; // [широта, долгота]
   const map = new ymaps.Map('yaMap', {
       center: coords,
       zoom: 16
   });
   
   const placemark = new ymaps.Placemark(coords, {
       balloonContent: 'Ваш адрес здесь',
       iconCaption: 'Ваш адрес здесь'
   });
   
   map.geoObjects.add(placemark);
   ```

## Как получить координаты адреса

1. Откройте Яндекс.Карты: https://yandex.ru/maps
2. Найдите нужный адрес
3. Кликните правой кнопкой мыши на точку
4. Выберите "Что здесь?"
5. В появившемся окне будут координаты в формате: `44.9511, 34.1024` (широта, долгота)

## Как работает карта

1. **API ключ**: В файле `index.html` подключен скрипт Яндекс.Карт с вашим API ключом:
   ```html
   <script src="https://api-maps.yandex.ru/2.1/?lang=ru_RU&apikey=91fc780b-6330-49af-b06a-6a0488240f20"></script>
   ```

2. **Геокодер**: Карта использует геокодер Яндекс.Карт для автоматического определения координат по адресу. Это означает, что вы просто указываете адрес текстом, а Яндекс сам находит координаты.

3. **Метка**: На карте автоматически ставится метка (маркер) в указанном месте.

4. **Fallback**: Если геокодер не сработает, используются резервные координаты `[44.9511, 34.1024]`.

## Параметры карты

- **zoom: 16** - уровень приближения (можно изменить от 1 до 19, где 1 - весь мир, 19 - максимальное приближение)
- **center** - центр карты (координаты или результат геокодирования)

## Примеры адресов

- `'Москва, Красная площадь, 1'`
- `'Санкт-Петербург, Невский проспект, 28'`
- `'Республика Крым, Симферопольский район, с. Мирное, ул. Центральная, 48'`

Чем точнее адрес, тем точнее будет метка на карте.
